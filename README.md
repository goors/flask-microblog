Flask microblog with mysql and sqlAlchemy
=========

## With this blog you can:

- login as admin
- crud posts
- crud users
- crud on comments
- add files to post
- add photo gallery for post
- add tags
- view posts by tags
- add comment(s)
    
    
## Requirements  

- Flask
- Pillow>=2.2.1
- pilkit>=1.1.5
- six>=1.4.1
- SqlAlchemy
- Mandrill
    
I used boostrap.css for responsive things and design.
